# Matrix Stats

Display the current Matrix memory statistics.

Use the `matrix_status` MCP tool to retrieve and present:

- **Solutions**: Total stored, by scope (global/stack/repo)
- **Failures**: Total recorded errors and fixes
- **Warnings**: File and package warnings
- **Usage**: Recent activity, success rates
- **Database**: Size and health

Format the output as a clear summary that helps the user understand their accumulated knowledge.
